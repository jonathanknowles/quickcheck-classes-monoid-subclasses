reductiveLaws :: forall a. (Arbitrary a, Show a, Eq a, Reductive a) => Proxy a -> Laws Source #

Laws for instances of Reductive.

Tests the following properties:

a </> b == stripPrefix b a

a </> b == stripSuffix b a

maybe a (b <>) (a </> b) == a

maybe a (<> b) (a </> b) == a

Note that the following superclass laws are not included:

  • commutativeLaws
  • leftReductiveLaws
  • rightReductiveLaws

Laws for instances of LeftReductive.
